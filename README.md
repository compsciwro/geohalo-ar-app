# AR Cosmic Quest

**AR Cosmic Quest** is a Unity-based AR mobile application that delivers a futuristic, neon-styled augmented reality experience.  
The app combines immersive visuals, interactive AR features, image tracking, ground plane interaction, customisation functionality, UI systems, audio, movement, and scene-based navigation into one complete mobile application.

---

## Overview

AR Cosmic Quest is designed as a full augmented reality application rather than a simple AR demo.  
It allows users to explore different AR experiences through multiple modes, each offering a different way to interact with digital content in the real world.

The app follows a cosmic sci-fi theme, with a glowing cyber-style interface and interactive AR scenes that create an engaging user experience.

---

## Features

- **Image Tracking Mode**  
  Uses image targets to detect and display AR content when the camera scans supported images.

- **Interactive Ground Plane**  
  Allows AR objects to be placed and interacted with on detected flat surfaces.

- **AR Customisation Mode**  
  Provides an interactive mode where users can engage with and customise elements of the AR experience.

- **Interactive 3D Content**  
  Includes 3D models, animations, and visual elements designed for AR interaction.

- **Scene Navigation**  
  Multiple scenes and menu navigation allow users to move between app experiences.

- **Movement and Interaction Systems**  
  Includes object/player movement and interaction logic built in Unity.

- **Collision and Feedback Systems**  
  Collision detection with audio and visual feedback.

- **UI Systems**  
  Includes interface elements such as menu buttons, health sliders, and visual HUD-style components.

- **Audio and Visual Effects**  
  Sound effects, particle systems, and animations help make the app feel more polished and immersive.

- **Android Mobile Build**  
  Packaged as a mobile AR application for Android devices.

---

## Tech Stack

- **Unity**
- **C#**
- **Unity XR / AR tools**
- **Unity Input System**
- **Unity UI**
- **TextMesh Pro**
